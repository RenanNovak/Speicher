# Speicher
  
### App de Tarefas

Tecnologias a ser utilizadas:<p>
-> Api JDBC para conexão com banco de dados (MySql)<p>
-> JavaFX para interface gráfica<p>
-> JUnit para realizar Testes Unitários<p>
